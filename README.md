# 🚗 Car Rental Application

Aplikasi Car Rental adalah sebuah aplikasi mobile Flutter yang dirancang untuk memudahkan pengguna dalam melakukan pemesanan kendaraan dengan antarmuka yang user-friendly dan fitur-fitur modern.

## 📋 Daftar Isi

- [Fitur Utama](#-fitur-utama)
- [Struktur Project](#-struktur-project)

## ✨ Fitur Utama

- ✅ Search daftar mobil
- ✅ Manajemen Mobil - CRUD mobil dengan filter & pencarian
- ✅ Pemesanan - Buat, kelola, dan batalkan pemesanan
- ✅ Pembayaran - Proses dan konfirmasi pembayaran

## 📁 Struktur Project

```
modul_6_car_rental/
```
